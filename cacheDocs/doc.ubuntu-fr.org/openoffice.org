<h1 class="sectionedit1" id="apache_openoffice_anciennement_openofficeorg">Apache OpenOffice anciennement OpenOffice.org</h1>
<div class="level1">

<p>
<img src="http://doc.ubuntu-fr.org/_media/openoffice_logo.png?w=200&amp;tok=9aa00b" class="mediaright" alt="" width="200" />
</p>

<p>
OpenOffice.org est un projet né le 13 octobre 2000 à l&#039;initiative de Sun Microsystems en vue de produire un intégré bureautique libre fondé sur StarOffice. La suite bureautique fut installée par défaut sur les anciennes versions d&#039;Ubuntu, avant l&#039;arrivée de Libreoffice. Le produit résultant est diffusé sous le même nom et sous plusieurs licences (la <abbr title="GNU Lesser General Public License">LGPL</abbr> et, jusqu’à la version 2.0beta2 incluse, la SISSL), et fonctionne sur plusieurs plates-formes dont Windows, de nombreux Unix : Linux, Solaris, ou Apple Mac <abbr title="Operating System">OS</abbr> X. Le but énoncé est d&#039;offrir une alternative à la suite bureautique propriétaire Microsoft Office à laquelle OpenOffice prendra une part de marché significative.
À partir de la version 3.3.0, le projet original s&#039;est scindé en deux branches distinctes :
</p>
<ul>
<li class="level1"><div class="li"> Apache OpenOffice, soutenu par la fondation <strong>Apache</strong>, c&#039;est de cette branche que traîte cette page ;</div>
</li>
<li class="level1"><div class="li"> <a href="libreoffice" class="wikilink1" title="libreoffice">LibreOffice</a>, soutenu par la fondation <strong>The Document Foundation</strong> et la communauté d&#039;utilisateurs d&#039;OpenOffice.</div>
</li>
</ul>

<p>
À noter aussi le fork allégé, totalement libre, basé sur openoffice.org, <a href="http://doc.ubuntu-fr.org/ooolight" class="wikilink1" title="ooolight">OOoLight</a>, créé par l&#039;association à but non lucratif EducOOo 
</p>

</div>
